## Overview

A self-initiated prototype that allows users to design, customize, and visualize interior layouts in a 3D environment. The app offers intuitive object manipulation, persistent storage, and sharing features for hobbyists or early interior planning stages.

## The Challenge

Mobile devices present unique challenges for 3D content creation — limited processing power, constrained input methods, and the need for responsive touch interfaces. The challenge was to create a design experience that felt natural on touchscreens while producing meaningful 3D output.

## Technical Approach

Developed touch-optimized input systems for object manipulation in 3D space. Implemented data serialization for design persistence and export capabilities for sharing.

## Key Implementations

- **Real-Time Layout Tools:** Placed, rotated, and scaled objects within a 3D space using touch gestures — pinch to scale, two-finger rotate, drag to position — for accurate visual feedback and arrangement flexibility.

- **Furniture & Object Catalog:** Included a varied asset set: sofas, lights, tables, kitchen units, decor, and appliances with category organization and search functionality.

- **Persistent Save/Load System:** Enabled users to store and reload design states, supporting multiple design variations and the ability to revisit and modify previous iterations.

- **Design Sharing:** Supported exporting design data to enable review, collaboration, or showcasing of designs with others through standard file formats.

- **Touch Interface Optimization:** Adapted traditional mouse-based design paradigms to touch input with gesture recognition and context-aware tool selection.

## Outcome

The prototype demonstrated that mobile devices could support meaningful 3D design experiences. Several UX patterns and interaction models from this project were adapted for commercial applications, influencing product decisions at partner organizations.
