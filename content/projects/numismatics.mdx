## Overview

Numismatics is a self-developed PC game built using DirectX 10, created as a technical showcase of procedural terrain generation and custom post-processing effects. The game challenges players to collect all scattered coins within a time limit across dynamically generated environments.

## The Challenge

Learning graphics programming from scratch requires understanding low-level rendering concepts that game engines typically abstract away. The challenge was to build a complete game while simultaneously implementing all rendering systems from first principles using DirectX 10 APIs.

## Technical Approach

Wrote low-level rendering logic for geometry, lighting, and effects pipelines directly in HLSL and C++. Implemented procedural terrain generation algorithms using noise functions for natural landscape creation.

## Key Implementations

- **Procedural Terrain Generation:** Developed a terrain system that creates unique landscapes each session using algorithmic noise functions and mesh generation with appropriate level-of-detail.

- **Gameplay Mechanics:** Designed core gameplay around time-based exploration and collection, encouraging spatial awareness and quick decision-making across procedurally varied terrain.

- **Post-Processing Effects:** Implemented real-time blur effects for camera transitions and motion emphasis using DirectX shaders and render targets.

- **DirectX 10 API Use:** Wrote low-level rendering logic for geometry, lighting, and effect pipelines directly in HLSL and C++ to build the engine from scratch, covering vertex buffers, index buffers, shaders, and render states.

- **Collision Detection:** Implemented terrain following and coin collection using custom collision detection optimized for procedurally generated geometry.

## Outcome

The project received the highest marks in the Programming Games module, demonstrating mastery of graphics programming concepts. Project documentation and code were subsequently used as reference material for later cohorts of students.
