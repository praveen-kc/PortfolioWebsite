## Overview

A technical prototype built using Unity's Mesh API to create fully dynamic room layouts at runtime based on user input. Designed to explore procedural geometry workflows and custom interaction systems within Unity, this project served as a sandbox for understanding low-level mesh generation and rendering pipeline concepts.

## The Challenge

Interior design applications require flexible room creation that feels intuitive but produces accurate, buildable geometry. Traditional approaches rely on pre-modeled assets, limiting user creativity. The challenge was to create a system where users could define arbitrary room shapes that translated directly into valid 3D geometry.

## Technical Approach

Implemented custom mesh generation algorithms that created wall, floor, and ceiling geometry from 2D user input. Built interaction systems for real-time shape editing and parameter adjustment.

## Key Implementations

- **Procedural Geometry Creation:** Generated runtime mesh data for walls, ceilings, and floors using raw vertex and triangle manipulation — no pre-made models used, all geometry created mathematically from user input.

- **Mouse-Driven Input System:** Allowed users to define room shapes using intuitive 2D mouse input, with automatic translation into accurate 3D space coordinates.

- **Shape Editing Tools:** Supported real-time resizing, corner dragging, and area redefinition post-creation for an iterative design workflow that didn't require starting over.

- **Surface-Type Differentiation:** Applied unique logic for vertical and horizontal surfaces to control textures, materials, and snapping rules appropriate to each surface type.

- **Error Handling & Validation:** Implemented geometry validation to prevent invalid room configurations like self-intersecting walls or impossible angles.

## Outcome

The techniques developed in this prototype directly influenced the architecture of later commercial interior design tools. The procedural approach to geometry generation became a recurring pattern in subsequent projects requiring dynamic 3D content creation.
