## Overview

Created as part of the Programming Games module during the master's program, this project focuses on implementing real-time lighting and surface interaction effects using DirectX 10. The scene demonstrates key rendering techniques aimed at enhancing realism through programmable shaders.

## The Challenge

Understanding modern rendering requires implementing fundamental algorithms from scratch. The challenge was to create a scene that clearly demonstrated multiple rendering techniques while maintaining visual appeal and real-time performance.

## Technical Approach

Implemented custom HLSL shaders for material response and lighting calculations. Developed multi-pass rendering pipelines for reflection and shadow effects using render targets and depth buffers.

## Key Implementations

- **Real-Time Reflection & Refraction:** Created surface shaders to simulate dynamic light bending and reflective materials using multiple render passes and screen-space techniques.

- **Dynamic Shadow Mapping:** Implemented depth-based shadows with accurate projection based on light direction and occluders, including shadow filtering for soft shadow edges.

- **Optimized Scene Composition:** Built a balanced 3D layout to clearly demonstrate the interaction of light, shadow, and surface materials with multiple material types and lighting conditions.

- **Shader Programming:** Wrote custom HLSL shaders for material response and per-pixel lighting accuracy, covering ambient, diffuse, specular, and emissive lighting models.

- **Performance Profiling:** Developed profiling tools to measure rendering performance and identify optimization opportunities in the shader pipeline.

## Outcome

The project was selected as an exemplary submission for the Programming Games module, with techniques incorporated into subsequent course materials. The implementation provided a foundation for understanding more advanced rendering concepts in later coursework and professional work.
