## Overview

Built as a console-level graphics programming experiment, this project showcases the integration of real-time fog simulation and lighting effects on PlayStation 2 hardware using the PS2 Linux development environment. It explores the limitations and capabilities of low-level graphics control on legacy consoles.

## The Challenge

Legacy console development presents unique challenges including limited documentation, constrained development tools, and hardware quirks not found in modern platforms. The PlayStation 2's unique architecture, with its separate vector and scalar units, required careful optimization. The challenge was to implement visually impressive effects within strict hardware constraints.

## Technical Approach

Programmed directly using the PS2 Linux SDK, focusing on graphics buffer management and performance tuning. Developed particle-based fog systems optimized for the console's vector unit capabilities.

## Key Implementations

- **Volumetric Fog Simulation:** Created a particle-based system to simulate atmospheric fog using minimal resources optimized for PS2 GPU constraints, achieving smooth visual results within tight budgets.

- **Directional Light Control:** Enabled adjustable lighting attributes such as rotation, intensity, and direction with real-time scene influence, demonstrating practical applications of the console's lighting capabilities.

- **Low-Level Graphics Pipeline:** Programmed directly using the PS2 Linux SDK, focusing on graphics buffer management and performance tuning to maximize the hardware's capabilities.

- **Hardware-Constrained Design:** Tailored visual logic and effects to accommodate PlayStation 2 memory and rendering limits, developing optimization techniques specific to the console architecture.

- **Debug & Visualization Tools:** Created in-game debug displays showing performance metrics and buffer utilization to understand hardware behavior during development.

## Outcome

The project pushed the boundaries of what was considered achievable on PS2 Linux, demonstrating that meaningful graphics experiments were possible without native development toolchains. Project documentation provided valuable insights into PS2 graphics capabilities for future students exploring legacy console development.
